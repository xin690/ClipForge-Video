**`category_match_score` 改法**:
```python
from core.synonyms import SynonymEngine

_SYN_ENGINE = SynonymEngine()
_SYN_ENGINE.load_yaml("data/synonyms.yaml")

def category_match_score(asset_tags, category_tags):
    if not category_tags:
        return 0.0
    atags = [t.lower() for t in asset_tags]
    hits = 0
    for cat_tag in category_tags:
        cat_lower = cat_tag.lower()
        if cat_lower in atags:
            hits += 1
        else:
            for atag in atags:
                if _SYN_ENGINE.score(cat_lower, atag) >= 0.4:
                    hits += 1
                    break
    return min(hits / len(category_tags), 1.0)
```

### 3.2 场景级效果

| 场景 | 当前 | 改后 |
|:---|:---|---:|
| 中文关键词 → 中文标签资产 | 0% 匹配 (无中文 CATEGORY_MAP) | ~60% 匹配 (F3+F4) |
| 英文关键词 → 中文标签资产 | ~20% 匹配 (纯碰文件名) | ~50% 匹配 (F3+F4+F7) |

| 情绪匹配 | 仅 TONE_MAP 精确 | +scene:high→strong, color:warm→happy (C1) |
